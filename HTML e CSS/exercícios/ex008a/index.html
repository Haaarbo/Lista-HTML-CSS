Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Já nesta outra frase, temos um termo em destaque usando a tag <strong> (semântica).

Qual a diferença?

A diferença está na SEMÂNTICA. Nesta versão do HTML5, a linguagem tem buscado o caminho de ser mais semântico, tem priorizado o significado da tag para com o texto do que sua forma.

Nos exemplos do começo, ambos possuem a mesma forma, já que ambas permitem o texto ficar em negrito; no entanto, enquanto a tag <b> foca apenas na forma, a de deixar em negrito, a <strong> traz um valor semântico, ela valoriza a ideia de dar destaque a algum texto, não se resumindo a simplesmente deixar o texto em negrito.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>.

Nesta frase temos um termo em ênfase usando a tag <em>.

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

Nesta frase também há um texto marcado pela cor ciano e outro pela cor carmesim.

É possível também usar o seguinte código em HTML dentro do <head> para customizar a cor de um mark:

<style>
mark{
background-color: _cor_
}
</style>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno, usando as tags <big> e <small> (ambas não são semânticas, mas apenas o <big> está em desuso por enquanto)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Este com a tag <del> (semântico).

Texto inserido

Podemos marcar um texto como inserido para indicar ênfase e indicar que ele foi adicionado posteriormente, usando a tag <ins> (semântico).

Há também como usar a tag <u> para a mesma função de marcar um texto como inserido, porém não carrega valor semântico e ,portanto, não é recomendado seu uso.
