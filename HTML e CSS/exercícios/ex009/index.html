Utilizando Listas

Listas Ordenadas

Estas podem ser ordenadas por letras maiúsculas ou minúsculas (A / a), por números romanos (I) ou por numeração comum (1).

Para definir quaisquer tipo, basta atribuir ao atributo "type" da tag <ol> o valor entre parênteses. Também pode ser definido seu início atribuindo a partir de qual item se iniciará a contagem para o atributo "start". A seguir alguns exemplos:

  1. Item 1
  2. Item 2
  3. Item 3

  2. Item 2
  3. Item 3

  2. Item 2
  3. Item 3

  2. Item 2
  3. Item 3

  2. Item 2
  3. Item 3
